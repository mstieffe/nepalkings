# Mixkit SFX Sources

The WAV masters in this directory are retained locally and Git-ignored. The
runtime derivatives are generated by `scripts/assets/generate_sfx.py`; raw
Mixkit downloads are never copied into a game or web bundle.

Runtime edits use 44.1 kHz PCM masters and Vorbis quality 6 web companions.
Spell cues retain their source stereo image; compact interaction cues are
mixed to mono.

Downloaded from the [Mixkit sound-effect library](https://mixkit.co/free-sound-effects/)
on 2026-07-13 under the [Mixkit Sound Effects Free License](https://mixkit.co/license/).
Mixkit permits its sound effects in commercial and personal projects and does
not require attribution. Nepal Kings nevertheless records each source, item
ID, runtime purpose, and checksum here.
